# 🏠 House Service

> Landing Page desenvolvida como projeto acadêmico da disciplina de **Desenvolvimento Front-End** do curso de **Engenharia de Software — UniCesumar**.

O **House Service** é uma plataforma web criada para conectar clientes e profissionais de serviços domésticos de forma simples, intuitiva e eficiente. O projeto tem como foco a construção de uma interface moderna, responsiva e de fácil navegação, proporcionando uma boa experiência para o usuário.

## 🌐 Demonstração Online

🔗 **Landing Page**

> * https://heloysaf.github.io/HOUSE-SERVICE-/*

## 💡 Sobre o Projeto

O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais do desenvolvimento front-end, explorando boas práticas de estruturação de páginas, estilização responsiva e interatividade utilizando JavaScript.

A aplicação apresenta uma landing page moderna para divulgação de serviços domésticos, destacando informações importantes de maneira organizada e acessível.

### Objetivos

- Desenvolver uma interface moderna e intuitiva;
- Aplicar conceitos de HTML, CSS e JavaScript;
- Criar um layout totalmente responsivo;
- Melhorar a experiência do usuário durante a navegação;
- Praticar técnicas de animação e interatividade.

---

## ✨ Funcionalidades

- 🧭 Navegação entre as seções da página;
- 📱 Layout totalmente responsivo;
- 📌 Barra de navegação dinâmica durante a rolagem;
- ✨ Animações de entrada dos elementos;
- 🎨 Interface moderna e organizada;
- 🚀 Experiência de navegação fluida e intuitiva.
